Create Traffic Matrix

This function overrides the same function of FlatNetwork The source demands is directed to the closest server.

Function Declaration

flow_list = CreateTrafficMatrix( this, source_demand_index)

source_demand_index: indicate which source_demand matrix is used for creating traffic matrix.

Find the closest server to the node

Useage: If one needs to select the nodes that do not pass a intermediate selected node, use the follwing code.

b = true(size(dest_node));
for j = 2:length(dest_node)
    for k = 1:j-1
        if ~isempty(find(path_list{j}, dest_node(k),1))
            b(j) = false;
            break;
        end
    end
end
dest_node = dest_node(b);
